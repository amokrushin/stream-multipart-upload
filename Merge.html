new Merge()

Merges array of objects to a single object

Example
new Merge()
  .on('data', (chunk) => {
    console.log(chunk);
  })
  .end([
    { metadata: { a: 1 }, errors: [new Error('a')] },
    { metadata: { b: 2 }, errors: [new Error('b')] },
    { metadata: { a: 3 } },
  ]);
